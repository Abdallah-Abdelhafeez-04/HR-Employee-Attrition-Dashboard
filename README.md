# HR-Employee-Attrition-Dashboard
Interactive HR Employee Attrition Dashboard built with Power BI to analyze employee turnover, workforce demographics, job satisfaction, and key HR KPIs using Power Query and DAX.

# HR Employee Attrition Dashboard

## Project Overview

This project presents an interactive HR Employee Attrition Dashboard developed in Microsoft Power BI.

The dashboard helps HR professionals understand employee turnover patterns by analyzing workforce demographics, job satisfaction, salary distribution, department performance, and employee attrition trends.

It transforms raw HR data into meaningful business insights that support data-driven decision making.

---

## Objectives

- Analyze employee attrition.
- Monitor HR KPIs.
- Understand workforce demographics.
- Compare departments.
- Identify factors influencing employee turnover.
- Support strategic HR planning.

---
